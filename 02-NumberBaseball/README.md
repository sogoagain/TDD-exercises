# 숫자 야구 게임

### 요구사항

1. 숫자 야구 게임
   - 정답인 세자리 숫자를 Player가 예상하여 맞추는 게임으로 한 번 시도할 때마다 정답인 숫자와 Player가 제시한 숫자의 비교 결과를 바탕으로 숫자를 맞춰나감
2. 게임 규칙
   - 사용되는 숫자: 0~9까지 서로 다른 숫자
   - 정답인 숫자 3자리를 Player가 맞춤
   - 볼: 숫자는 맞고 위치는 틀림
   - 스트라이크: 숫자와 위치 모두 맞음
   - 아웃: 숫자와 위치 모두 틀림
   - 제시한 숫자 중 어떤 숫자가 볼/스트라이크/아웃인지 알려주지 않음
